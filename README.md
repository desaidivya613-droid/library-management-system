# LIBRARY MANAGEMENT SYSTEM (MySQL)

# DESCRIPTION
This project is a MySQL-based Library Management System that manages books, members, and issued books.

# FEATURES
- Add and manage members
- Add and manage books
- Issue and return books
- Track overdue books
- Generate reports using SQL queries

# TTECHNOLOGIES USED
- MySQL

# FILES
- library.sql → Contains complete database and queries

# LEARNING OUTCOMES
- Learned SQL queries (CRUD operations)
- Used JOINs and relationships
- Designed relational database system
